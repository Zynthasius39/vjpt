## Deployment

### Install dependencies
```$ npm install```

```$ bun install```

### (Optional) Run development server
```$ npm run dev```

```$ bun run dev```

### Build
```$ npm run build```

```$ bun run build```

### Start your choice of webserver in `/dist`
```$ python3 -m http.server -d ./dist 8000```
